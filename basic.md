# Design Pattern



0. 设计模式存在的意义？

        设计模式是对软件设计中普遍存在(反复出现)的各种问题,所提出的解决方案,
        是从建筑设计领域引入到计算机科学的, 经过仔细设计后搭建的项目具有更好的高内聚性, 低耦合性, 重用性, 稳定性, 可拓展性, 易维护性, 可读性, 规范性等等


1. 设计模式在软件中哪里?
    
        面向对象(oo) => 
            功能模块(设计模式 + 数据结构/算法) => 
                框架(多种设计模式) => 
                     框架(服务器集群)
